# 说明

这是一个 PySide6 应用模板，重点在于展示如何支持多语言。

## 注意

.ts 文件中的词条包含行信息，也就是说如果在生成了 .ts 文件后又修改了 .py 文件，导致词条所在的行号不一致了，那么就会翻译失败。这时需要重新生成 .ts 文件，重新发布 .qm 文件，重新转换 .qrc 文件。
